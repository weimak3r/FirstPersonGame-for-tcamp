Gameplay

核心玩法：双人合作对抗 AI 敌人

玩家

WASD 移动，鼠标控制视角，左键射击。HUD 显示血量、分数和胜利提示。

敌人

敌人自动寻找最近玩家，持续靠近并攻击。被击杀后同步消失。

目标

玩家合作击杀敌人获得分数，达到目标分后触发胜利。

Demo 重点是功能完整和网络同步，不追求复杂美术。
Player System

玩家系统

  • Enhanced Input：移动、视角、跳跃、射击。
  • 第一人称摄像机 + 枪模型占位。
  • 射击使用服务端射线检测。
  • 蓝色短射线用于表现弹道。
  • 血量通过复制同步到客户端。
RPC

客户端按下左键后调用 Server RPC，由服务端进行命中和伤害判定，避免客户端各自计算导致不同步。

关键类：FPCharacter / FPPlayerController / FPGameHUDWidget

敌人 AI 与死亡同步

Score & Reset

得分、胜利与软重置

得分

击杀敌人的玩家获得分数，分数保存在 GameState 中。

胜利

分数达到目标后通知对应客户端显示 YOU WIN。

软重置

按 R 不重载地图；服务端重置玩家、敌人、分数和 HUD，避免多人世界线分裂。

关键类：FPGameState / FPGameMode / FPCharacter
