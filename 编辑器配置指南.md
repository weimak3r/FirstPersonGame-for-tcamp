# UE5.4.4 编辑器配置操作文档

> 适用版本：Epic Games 官方 UE5.4.4
> 前提：双击 `.uproject` → 弹窗点 **Yes** 编译通过，进入编辑器界面

---

## 一、创建 Input Actions（按键资产）

### 1.1 创建 IA_Move

1. 底部 **Content Browser**（内容浏览器）→ 左侧找到 `Content` 文件夹
2. 右键空白处 → **Miscellaneous（杂项）** → **Input Action**
3. 命名为 `IA_Move`
4. 双击打开 → **Value Type** 设为 `Axis2D (Vector2D)`
5. 关闭窗口

### 1.2 创建 IA_Look

1. 同上步骤创建 `IA_Look`
2. **Value Type** 设为 `Axis2D (Vector2D)`

### 1.3 创建 IA_Jump

1. 创建 `IA_Jump`
2. **Value Type** 设为 `Digital (bool)`

### 1.4 创建 IA_Shoot

1. 创建 `IA_Shoot`
2. **Value Type** 设为 `Digital (bool)`

---

## 二、创建 Input Mapping Context（按键映射）

### 2.1 创建 IMC_Default

1. 右键 Content 空白处 → **Miscellaneous** → **Input Mapping Context**
2. 命名为 `IMC_Default`
3. 双击打开

### 2.2 绑定按键

| Input Action | 按键 | 修饰器（Modifiers） |
|-------------|------|-------------------|
| IA_Move | W | 无 |
| IA_Move | S | Negate |
| IA_Move | A | Swizzle Input Axis Values（YXZ） |
| IA_Move | D | Swizzle Input Axis Values（YXZ）+ Negate |
| IA_Look | Mouse XY 2D-Axis | 无 |
| IA_Jump | Space Bar | 无 |
| IA_Shoot | Left Mouse Button | 无 |

**操作方式：**
- 点击 **+** 添加 → 搜索选择 Input Action → 展开下拉选按键
- 修饰器点 **+** 搜索添加

---

## 三、创建蓝图子类

### 3.1 创建 BP_FPCharacter（玩家角色）

1. 右键 Content → **Blueprint Class**
2. 搜索父类 `FPCharacter`，选中 → 命名为 `BP_FPCharacter`
3. 双击打开蓝图编辑器
4. 右侧 **Details（细节面板）** → 搜索 `Default Mapping Context`
5. 下拉选择 `IMC_Default`
6. 同理设置 `Move Action` → `IA_Move`，`Look Action` → `IA_Look`，`Jump Action` → `IA_Jump`，`Shoot Action` → `IA_Shoot`
7. **Compile（编译）** → **Save（保存）**

### 3.2 创建 BP_FPPlayerController

1. 新建 Blueprint Class → 父类 `FPPlayerController`
2. 命名为 `BP_FPPlayerController`
3. 先不管（HUD Widget Class 等创建 WBP 后再填）

### 3.3 创建 BP_FPGameMode

1. 新建 Blueprint Class → 父类 `FPGameMode`
2. 命名为 `BP_FPGameMode`
3. 右侧 Details → **Default Pawn Class** → 选 `BP_FPCharacter`
4. **Compile → Save**

### 3.4 创建 BP_FPEnemy（敌人）

1. 新建 Blueprint Class → 父类 `FPEnemy`
2. 命名为 `BP_FPEnemy`
3. 打开蓝图 → 左侧 **Components（组件面板）**
4. 点击 **Add Component（添加组件）** → 搜索 `Static Mesh`
5. 选中添加的 Static Mesh → 右侧 Details → **Static Mesh** → 下拉选 `Shape_Cylinder`（临时占位用，后面可换成角色模型）
6. 同样添加 `Capsule Component`（碰撞体，如果继承的 Character 没有的话通常已有）
7. **Compile → Save**

## 四、创建 HUD Widget 蓝图

### 4.1 创建 WBP_GameHUD

1. 右键 Content → **User Interface** → **Widget Blueprint**
2. 父类选 `FPGameHUDWidget` → 命名为 `WBP_GameHUD`
3. 双击打开 Widget 编辑器

### 4.2 搭建 UI 布局

左侧 **Palette（控件面板）** 拖入以下控件到 **Hierarchy（层级）** 根下：

| 控件类型 | 命名 | 用途 |
|---------|------|------|
| Canvas Panel | CanvasPanel_Root | 根容器（通常已有） |
| Progress Bar | HealthBar | 血条 |
| Text Block | ScoreText | 分数 |
| Text Block | VictoryText | 胜利文字 |
| Image | Crosshair | 准星 |

### 4.3 设置默认值

- **VictoryText**：Visibility 设为 `Hidden`，Text 清空
- **Crosshair**：设置一张准星图片（可先用白色圆形占位），锚点设为屏幕中心
- **HealthBar**：Percent 设为 `1.0`

### 4.4 绑定名称

在 Hierarchy 中选中控件 → 右侧 Details 顶部 → 确认 **Is Variable** 勾选。

> 注意：控件名必须和 C++ 代码中 `meta = (BindWidget)` 的名字完全一致：
> `HealthBar`、`ScoreText`、`VictoryText`、`Crosshair`

### 4.5 Compile → Save

## 五、回填引用

### 5.1 BP_FPPlayerController 设置 HUD

1. 重新打开 `BP_FPPlayerController`
2. 右侧 Details → **HUD Widget Class** → 选 `WBP_GameHUD`
3. **Compile → Save**

### 5.2 项目全局设置

1. 顶部菜单 → **Edit（编辑）** → **Project Settings（项目设置）**
2. 左侧选 **Maps & Modes（地图 & 模式）**
3. **Default GameMode** → 选 `BP_FPGameMode`
4. **Default PlayerController Class**（可能有，搜索一下）→ 选 `BP_FPPlayerController`
5. 关闭窗口

## 六、关卡搭建

### 6.1 打开默认关卡

1. **File → New Level → Basic**（或打开已有的默认关卡）
2. 如果默认关卡是模板带的，Content 里找 `FirstPersonMap`（没有就新建）

### 6.2 放置玩家出生点

1. 左侧 **Place Actors（放置Actor）面板** → 搜索 `Player Start`
2. 拖入关卡视口，放到地面上

### 6.3 放置敌人

1. Place Actors → 搜索 `BP_FPEnemy`（可能要等一会索引）
2. 拖入关卡 → 放到离玩家出生点有一段距离的位置
3. 多拖几个（建议 6-8 个），分散放置

### 6.4 添加 NavMesh（导航网格——敌人移动需要）

1. Place Actors → 搜索 `Nav Mesh Bounds Volume`
2. 拖入关卡 → 用缩放工具拉大到覆盖整个游戏区域
3. 按键盘 **P** 键——绿色区域显示 = NavMesh 生效

### 6.5 添加地面

1. Place Actors → **Shapes → Cube**
2. 拖入关卡，Scale 拉大做地面（比如 Scale X=20, Y=20, Z=0.1）

## 七、运行测试

1. 顶部工具栏点击 **Play（播放按钮）** → 或快捷键 **Alt+P**
2. 预期效果：
   - WASD 移动，鼠标转视角
   - 左键射击 → 打中敌人 → 敌人消失 → 分数 +10
   - 杀够 5 个（50 分）→ 屏幕显示 **YOU WIN!**
3. 按 **Esc** 退出

### 7.1 测试多人联网

1. Play 按钮旁下拉 → **Number of Players** → 设为 2
2. 启动后有两个窗口 → 一个作为 Host，一个作为 Client
3. 两个窗口都能看到分数同步更新

## 八、常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| 编译失败 | C++ 代码有语法错误 | 检查 Output Log（Window → Developer Tools → Output Log） |
| 按键没反应 | IMC 未关联到角色 | 回第三步检查 BP_FPCharacter 的 DefaultMappingContext |
| 敌人不动 | 没放 NavMesh | 回 6.4 加 NavMesh Bounds Volume |
| HUD 不显示 | HUDWidgetClass 未设 | 回 5.1 检查 BP_FPPlayerController |
| 点 Play 报错 | 未设 GameMode | 回 5.2 检查 Maps & Modes |
